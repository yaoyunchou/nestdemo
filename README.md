## 项目介绍

项目基础技术栈是：Nestjs 9.x + TypeORM + MySQL5.7



运行项目之前，请先自行配置本地的数据库环境，并修改项目根目录中的配置文件：`.env`，`.env.development`（开发），`.env.production`（生产）。



推荐使用`node14 LTS`安装依赖：

```
npm i 
```



运行项目：

```
npm run start:dev
```
